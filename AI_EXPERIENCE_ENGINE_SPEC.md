# AI Workspace 「知的体験デザイン・AIエクスペリエンスエンジン（AI Experience Engine）」機能設計仕様書

本書は、AI Workspaceにおけるすべてのユーザー接点（インターフェース）において、世界最高峰の認知工学、デザインシステム、動的なマイクロモーション、利用文脈アライン型レイアウトを調和させ、ユーザーに極上の「直感性」「没入感」「高級感」「未来感」を提供する次世代体験創出・フロントエンド統治プラットフォーム**「AI Experience Engine（AIエクスペリエンス・エンジン）」**の意匠設計、システム構成、データスキーマ、適応型レイアウト論理（Adaptive UI）、および画面レイアウトを定義した製品設計仕様書です。

本システムは、単なる静的で画一的なUIテンプレートを完全に廃し、AIがユーザーのその瞬間の利用状況（ワークロード、時間帯、経営的優先度、デジタルツインの稼働状態）を瞬時に自律解析し、ホーム画面・メニュー・ショートカット・視覚ビジュアライザ（図、タイムライン、フローチャート、マップ、マインドマップ）を動的に生成・リサイズ・再配置。さらに、目標達成や成果創出の瞬間にエモーショナルで高揚感あふれる「感動演出（Emotional Design）」を適用する、AI Workspace全体の**「自律最高デザイン・体験・ブランド責任者（Autonomous CDO / Chief Experience Officer Engine）」**の役割を果たします。

## 3. デザインシステム ＆ ブランドアイデンティティ定義 (Design System & Brand Rules)

AI Experience Engineのすべてのコンポーネントが、世界に類を見ない「美しさと整合性」を保てるよう、厳密に体系化された「ラグジュアリー・知的デザインシステム」のコード規約。

### ① 統一カラーパレット (Color Archetype)
神秘的で目の疲れない深淵なダーク基調に、本物の貴金属を思わせる「ゴールド」および「エメラルドグリーン」を極めて限定的に配置し、圧倒的な高級感を表現します。

*   **Background (深淵):** `Carbon Black` (`#09090b` / `rgba(9,9,11,1)`)
    *   画面全体のベース背景。漆黒の中に、宇宙の奥行きを感じさせる極小のノイズグレインを付与。
*   **Card / Panel (半透明アクリル):** `Graphene Slate` (`rgba(20,20,23,0.75)`)
    *   グラスモーフィズムを適用。背面ブラー（`backdrop-filter: blur(24px)`）と、1pxの微細な境界線（`rgba(255,255,255,0.08)`）で浮かび上がらせる。
*   **Aesthetic Accents (金銀・エメラルド・ミント):**
    *   `Royal Gold` (`#d4af37` / `#f3e5ab` へのグラデーション) ➔ 最重要の成果達成（Celebration）、Golden Answerの冠、マスターシンボルに使用。
    *   `Intelligence Blue` (`#00f0ff` / `#0072ff`) ➔ AIの思考、データハイドレーション、並列実行の接続線に使用。
    *   `Outcome Mint` (`#2ec4b6`) ➔ コンバージョン達成、目標値クリアのグリーンインジケータ、実行承認ボタンに使用。
*   **Text Hierarchy (タイポグラフィ配色):**
    *   `Primary Title`: `#fafafa` (完全な白ではなく、目に優しいオフホワイト)
    *   `Secondary Text`: `#a1a1aa` (上品なマットグレー)
    *   `Muted/Mono Label`: `#71717a` (控えめなダークグレー、コード・メタデータ用)

### ② タイポグラフィ・フォント選定 (Typography Pairing)
*   **Display & Title (英数字):** `Space Grotesk` (Google Fonts)
    *   幾何学的で未来感にあふれ、知的で引き締まったトラッキング（`tracking-tight`）を設定。
*   **Body & Japanese (日本語):** `Inter` + `Noto Sans JP` (Google Fonts)
    *   抜群の視認性と読みやすさを誇り、長いドキュメントでも目の疲労を最小限に抑える。
*   **Metadata & Data (コード・実績値):** `JetBrains Mono` (Google Fonts)
    *   統計値、時間、相関係数（r=0.82）、TypeScriptコードブロックを、知的でプロフェッショナルな「エンジニアリング意匠」に昇華。

### ③ インタラクティブ・モーション ＆ マイクロインタラクション (Motion Rules)
すべての遷移、ローディング、ボタンホバー、およびAI生成演出には、物理法則（スプリング・ダンピング係数）に基づいた滑らかなイージングを使用し、チープな「フェードイン」や「遅延」を排します。

*   **Spring Animation (motion/react 用設定):**
    *   `stiffness: 180`, `damping: 24`, `mass: 1` ➔ 弾みすぎず、しっとりと吸い付くような高級車のサスペンションを模したモーショングリッド。
*   **AI Thinking演出 (思考の光):**
    *   AIが思考中（1.25秒間）は、カードの1pxの境界線が「Intelligence Blue」から「Royal Gold」へ、ニューラルネットワークのパルス信号のように高速で流れるグラデーションアニメーション（`animate-shimmer`）を適用。
*   **セマンティック・マージ演出 (Flicker-Free Merge):**
    *   複数のAIから回答が統合される瞬間、テキストがバラバラに表示されて画面がガタつく「フリッカー（ちらつき）」を完全に回避。透明度0%から100%へ、スケール（0.98 ➔ 1.0）とブラー（10px ➔ 0px）を伴って 250ミリ秒で滑らかに出現。

### テーブル定義詳細

#### ① `experience_profiles` (エクスペリエンス・プロファイル)
ユーザーごとの視覚・動作のパーソナライゼーション設定、テーマ、モーション感度、および同期されたレイアウト状態を統括。
*   `id`: UUID (Primary Key)
*   `company_id`: UUID (Foreign Key - 組織スペースのID)
*   `user_email`: VARCHAR(255) (ユーザーのメールアドレス。例: `nori72ny@gmail.com`)
*   `active_theme_name`: VARCHAR(100) (適用中の意匠テーマ。`CARBON_GOLD_LUXURY` (標準マテリアルカーボン), `NEON_CYBER` (サイバーパンク), `ALABASTER_LIGHT` (超高画質ライト))
*   `motion_sensitivity`: DECIMAL(3, 2) (アニメーション速度係数。`1.00` (標準), `0.00` (アニメーション完全無効・低負荷), `0.50` (高速シャープ))
*   `visual_quality_level`: VARCHAR(50) (ビジュアル描画品質。`WEBGL_HIGH` (3Dマインドマップ・Webマウント), `STANDARD_CSS` (2DフラットCSS))
*   `last_synced_layout_json`: JSON (次回ログイン時に前回の完璧なデスクトップ配置を復元するためのグリッド構成キャッシュ)
*   `created_at` / `updated_at`: TIMESTAMP

#### ② `experience_layouts` (適応型レイアウト配置テーブル)
AIがユーザーの行動・職務（経営者、開発者など）に応じて自動的に並び替えた、各ウィジェット（タスク、BIグラフ、AI チャット等）の二次元座標およびリサイズ比率。
*   `id`: UUID (Primary Key)
*   `profile_id`: UUID (Foreign Key)
*   `view_name`: VARCHAR(100) (表示している画面名。例: `WORKSPACE_HOME`)
*   `widget_id`: VARCHAR(100) (ウィジェット名。例: `TODAY_PRIORITIES`, `BI_REVENUE_CHART`, `AI_CHAT_STUDIO`)
*   `grid_x` / `grid_y`: INT (グリッド上の配置開始座標)
*   `width` / `height`: INT (グリッド上の幅と高さ。例: `width=12, height=4` はフルスクリーン横幅)
*   `is_visible`: BOOLEAN (現在AIによって表示状態と判定されているかどうか)
*   `created_at`: TIMESTAMP

#### ③ `experience_shortcuts` (自律パーソナライズ・メニュー)
ユーザーの過去の頻繁な利用、または現在の重要プロジェクト（PJ-204）から、AIが「今アクセスすべき」と推定して左レール等に自動配備した重要動的ショートカット。
*   `id`: UUID (Primary Key)
*   `profile_id`: UUID (Foreign Key)
*   `label`: VARCHAR(255) (メニュー表示名。例: `「新宿店予約LP(V3.2)確認」`)
*   `target_view`: VARCHAR(255) (遷移先ビューパスまたはアクションID)
*   `weight`: DECIMAL(5, 4) (アクセス頻度およびコンテキスト予測に基づく重要度スコア。高いものから順にTop 5を自動表示)
*   `is_pinned`: BOOLEAN (ユーザーがピン留めして固定したかどうか)
*   `created_at`: TIMESTAMP

#### ④ `experience_celebrations` (感情演出・アチーブメント実行ログ)
目標達成や成果創出の瞬間に、どのようなシネマティック演出（3Dゴールドシャワー、進捗達成ドラム等）が何時実行されたかの記録。
*   `id`: UUID (Primary Key)
*   `profile_id`: UUID (Foreign Key)
*   `event_type`: VARCHAR(100) (演出イベント。`GOAL_COMPLETED` (目標達成), `MERGE_COMPLETED` (共同制作完了), `SYSTEM_ONLINE` (デプロイ成功))
*   `payload_json`: JSON (演出に必要なメタデータ。`{"confetti_colors": ["#d4af37", "#2ec4b6"], "message": "新宿店 Stripe 売上 ¥1,500,000 目標突破！"}`)
*   `executed_at`: TIMESTAMP

#### ⑤ `experience_visual_renderings` (認知ビジュアル出力キャッシュ)
回答テキストから「フロー」「タイムライン」「マインドマップ」へ自動パースされた、Recharts や D3.js 用の構造データ。
*   `id`: UUID (Primary Key)
*   `profile_id`: UUID (Foreign Key)
*   `query_id`: UUID (Foreign Key - 該当する質問・回答ログのID)
*   `render_type`: VARCHAR(50) (描画形式。`MAP` (ロケーション地図), `FLOW` (フローチャート), `TIMELINE` (ガント・タイムライン), `MINDMAP` (3Dマインドマップ), `COMPARE_TABLE` (比較表))
*   `source_code`: TEXT (ビジュアライザコンポーネントが動的に読み込んでレンダリングするためのJSON・ Mermaid / D3コード)
*   `created_at`: TIMESTAMP

### ① ビジュアル構造のセマンティック抽出アルゴリズム (Visual Auto-Parser)
AIの回答 $A$ から、最適なレンダリングビジュアル $V_{type}$ を決定するルールベース ✕ セマンティック確信度モデル。
*   **IF** $A$ contains `["1位", "2位", "相関係数", "売上", "MEO"]` ➔ `render_type = 'CORRELATION_GRAPH'` (Recharts時系列相関)
*   **IF** $A$ contains `["最初", "次に", "その後", "➔", "ステップ", "フロー"]` ➔ `render_type = 'FLOW_CHART'` (D3.js フロー)
*   **IF** $A$ contains `["マイルストーン", "日程", "〜日", "スケジュール"]` ➔ `render_type = 'TIMELINE_GANTT'` (D3 Timeline)
*   本システムは、ユーザーが文章を「読む」ストレスを完全に排除し、情報はすべて「直感的に見て操作する（Visualized Canvas）」形で眼前にマウント配置します。

    *   Truth Engine によって100%嘘やハルシネーションがないと実証（Verified）されたデータカードには、カード上部に「👑 **TRUTH VERIFIED (証拠スコア: 100%)**」という、Royal Gold に輝く微細なグラデーションバッジを自動マウント。
