## 1. 哲学 (Cognitive Philosophy)

> **AIは単に言葉（回答）をその都度生成する静的エンジンではない。**  
> **AIとは、有機的に思考し、自己適応し続ける「躍動する巨大な電子脳」である。**

AI Workspaceのコグニティブ・コアには、人間の世界一流スペシャリストチームが互いに強固に協調し、高め合うような、多重構造のニューラルマトリクス**「Digital Brain (デジタルブレイン)」**が搭載されています。

私たちは、単一のプロンプトを単一のLLMに投げて終えるような浅薄な処理を認めません。1つのインプットに対し、10の階層化された「知能ブレイン（Layers of Brain）」がミリ秒単位で協調・並列処理を行い、人間を超える多角的なアプローチで「最高の結果（成果）」を自律的に導出します。

### Layer 1: Intent Brain (目的理解脳)
ユーザーの「言葉の字面」ではなく、「暗黙の真の目的」を完全に看破し、必要な全要求を自律展開するレイヤー。
*   **動作メカニズム**:
    -   ユーザーが「ホームページを作って」と一言入力した瞬間、Intent Brainは即座にビジネスゴールを逆算・パースします。
    -   ➔ `[売上最大化]` `[集客力の獲得]` `[競合を凌駕するデザイン]` `[SEO・MEO・AIO（AI検索最適化）の同時ビルド]` の4つの隠れた目的（Latent Goals）を自動で言語化し、全体のタスク（DAG）に注入します。

### Layer 2: Knowledge Brain (知識脳)
社内データ、Web検索（グラウンディング）、最新学術論文、各社公式仕様API、および過去の全プロジェクト履歴を一元的にマージし、知の基盤（Knowledge Base）を構成するレイヤー。
*   **動作メカニズム**:
    -   指示を受け取ると、社内のプライベートセマンティックインデックスを走査しつつ、リアルタイムWeb検索（Google Search API連携）を実行。
    -   過去の成功したLPのスタイルやコードアセット（Memory Ledger）も自動でロードし、完璧な最新情報を推論チームに引き渡します。

### Layer 3: Reasoning Brain (推論脳)
複数の異なる最高峰AIモデル（Gemini 2.5 Pro vs GPT-4o vs Claude 3.5 Sonnet）を動的にアロケーションし、並列議論・仮説検証・論理矛盾の潰し込みを行う思考の主柱レイヤー。
*   **動作メカニズム**:
    -   作成されたドラフトに対して、「論理に一貫性はあるか」「エッジケースでの不具合はないか」をAI同士で徹底議論（Debate Protocol）。
    -   相互の弱点やUX上の懸念を完全に叩き潰した「最高の一手」が合意されるまで、ミリ秒での自己リフレクションを繰り返します。

### Layer 4: Truth Brain (真実脳)
ハルシネーション（AIの嘘）を絶対防衛するための多重監査（Fact-checking Bulwark）を実行するレイヤー。
*   **動作メカニズム**:
    -   提出前に、公式情報の有無、最新ニュースとの不整合、複数AI間の一致率（Consensus Index）、提示された数値の計算正確性、引用元のURL生存性、論理的矛盾、確信Logprobsをすべてスキャン。
    -   嘘や未確認の主張を100%排除し、すべてのファクトにスマートな脚注アンカー（[1],[2]）を自動マッピングします。

### Layer 5: Creation Brain (制作脳)
テキストによるアドバイス（口頭での解説）で終わらせず、動作する実用的な成果物をその場で自動ビルド（Production Factory）するレイヤー。
*   **動作メカニズム**:
    -   仕様、デザイン美学（Apple HIG/Stripe/Notion）に適合した、稼働React/TypeScriptコード、高精細ベクターイラスト、PDF資料アウトライン、AIO用構造化JSON-LDデータを裏側で自律書き出し。
    -   ユーザーが「そのまま仕事にコピペ、またはデプロイして使える」極上クオリティで物資を製造します。

### Layer 6: Quality Brain (品質脳)
完成した成果物の商業利用適正を、厳格な100点満点の評価ルーブリックで自己採点し、品質の最低基準を冷徹に監視するレイヤー。
*   **動作メカニズム**:
    -   正確、網羅、専門、UX、セキュリティ等の項目から得点（Score）を算出。
    -   **「品質95点未満は、ユーザーの画面に絶対に提出しない（提出禁止）」**という鉄の掟に従い、95点未満の初稿は自動でExcellence Loop（リファクタリング差し戻し）へ送られ、**99点以上の「神の完成度」**に磨き上げられた段階で提出承認（Pass）を下します。

### Layer 7: Experience Brain (体験脳)
難解なデータや膨大な情報をそのままユーザーにぶつけず、脳が最も心地よく瞬時に理解できる「ビジュアルファースト」のレイアウトに超高速パッキングするレイヤー。
*   **動作メカニズム**:
    -   逆ピラミッド構造に基づき、最上部には「directAnswer（10秒の結論）」と「品質/合意率インジケータ」を配置。
    -   その下部に、情報カード、マトリクス比較表、Rechartsデータグラフ、SVGによる自律図解、および時間のタイムラインを整理配置し、スクロールだけで理解が完結するインタラクティブステージを構成します。

### Layer 8: Future Brain (未来予測脳)
今この瞬間の課題解決（TODO）だけに囚われず、ユーザーが次の半年後、一年後に直面するであろう戦略的課題、必要になる機能、追加アセットを自律シミュレーションするレイヤー。
*   **動作メカニズム**:
    -   「ホームページ作成」タスク完了時に、「店舗集客が3ヶ月後に急増することから逆算し、夜間自動返信ボットと、Stripe対応型決済フォームのワークフロー」を先回り予測（Future Prediction）。
    -   ユーザーに未来の成長ロードマップと、そのプロトタイプを同時提案します。

### Layer 9: Memory Brain (記憶脳)
ユーザーとの永続的な関係を築き、日々完全にフィットした「あなた専用の知的OS」へと自己最適化し続けるレイヤー。
*   **動作メカニズム**:
    -   利用履歴、ユーザーが過去に承認（Approve）したデザインの余白比率・トーン、嫌う言葉遣い、役職、ビジネスドメインを「User DNA Schema」に自動コンパイル。
    -   日を追うごとに、暗黙の前提（阿吽の呼吸）が完璧に通じる「もうひとりの自分」へと自己適応し続けます。

### Layer 10: Master Brain (司令部)
他のすべてのブレイン（Layer 1〜9）を統制・調停・プロキシし、タスクごとに最適なモデル、最適なワークフロー（DAG）、最適な品質、最適な外部連携コネクタを瞬時に差配する「最高意思決定知能」。
*   **動作メカニズム**:
    -   オーケストレーターの中枢として機能。システム全体のエラー復旧や、APIタイムアウト時の無言の自己修復（Silent Repair）、トークン消費量の動的キャッピングを一元的にコントロールし、100%の確率で「成功」を届けます。
