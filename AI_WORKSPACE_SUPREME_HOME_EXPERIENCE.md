# Experience Bible - Chapter 01: Supreme Home Experience (至高のホーム体験)
**Classification:** Product Experience Blueprint / Supreme Interface Specifications  
**Version:** 1.0.0  

---

## 1. 哲学 (Design Philosophy)

> **ホーム画面は単なる「機能の入り口」ではない。**  
> **起動した瞬間に未来を確信させ、説明なしで脳と同期する「知的共鳴のステージ」である。**

従来のAIプラットフォームやチャットボットは、画面中央に寂しく佇むインプットボックスと、左側に積み重なる時系列の会話ログという、退屈で画一的なデザインテンプレートに縛られていました。これらは単なる機能のパッチワークであり、インスピレーションを刺激することも、ユーザーを自律的な成功へ導くこともありません。

**Supreme Home Experience（至高のホーム体験）**は、Appleの洗練された静寂、Teslaの緻密な操作制御、Arcブラウザの境界をなくす滑らかさ、Notionの直感的な情報階層、Linearの圧倒的なスピード、Figmaの無限のクリエイティビティ、そしてCursorの統合的思考力をすべて融合し、凌駕する体験を追求します。

それは、1画面という極めてコンパクトな制約のなかで、美・知能・スピードが完全に調和し、ユーザーが思考を吹き込んだ瞬間にすべてのアセットが自律形成される、まさに「未来のオペレーティングシステム」の顔となるホーム体験です。

## 2. 7つの体験原則 (Experience Core Principles)

どのAIよりも美しく、分かりやすく、早く、未来を感じさせるため、ホーム画面の設計および動作に以下の7つの鉄則を課します。

1.  **直感の同期 (Self-Explanatory & Pathless)**:  
    説明書、チュートリアル、ヒントガイドなどの無駄なノイズを一切排除。見た瞬間に何をすべきかが本能レベルで理解でき、一切迷わせないゼロ摩擦のUI。
2.  **言葉を超える（Proactive Ingestion）**:  
    単なる「検索窓」ではなく、ユーザーの入力を待ち受ける瞬間に、周囲の脈絡や過去のコンテキストから「真に求めている潜在目標」を自律的に先回りして予測表示（Elicits more than asked）。
3.  **静的から動的へ（Organic Pulsing UI）**:  
    画面全体が死んだ静的なモックではなく、AIが背後で思考・検索・議論している様子を、微細で心地よいアンビエントな明滅（Ambient Pulse）やグラデーションウェーブで表現。
4.  **100ミリ秒の絶対規律（The 100ms Rule）**:  
    すべてのトランジション、クリック、展開アニメーションは、知覚的な遅延をゼロにするため100ms〜250msの超高速イージングで処理。
5.  **カード・ファースト構造（Visual Capsule）**:  
    すべての情報は剥き出しのテキストではなく、贅沢な知的余白（Intellectual White Space）に縁取られた「情報カプセル（Polished Cards）」として独立させ、視認性と美的美観を最大化。
6.  **レスポンスの段階的開示（Progressive Disclosure）**:  
    一度にすべての複雑な設定や情報を表示して認知負荷を高めることを頑なに拒否。基本は極限までミニマルにし、ユーザーがフォーカスした（ホバー、クリックした）ディテールのみを滑らかに拡張展開。
7.  **完全レスポンシブ同期（Device Sovereignty）**:  
    デスクトップ、モバイル、タブレットのいかなる画面アスペクト比においても、要素がただ潰れて並び変わるのではなく、その解像度で「最も美しい視覚的黄金比」を維持して自動リサイズ。

## 3. 画面構成 ＆ 1画面レイアウト (The Single-View Stage Blueprint)

ホーム画面は、無駄な複数遷移や階層深いサイドナビゲーションを廃し、**「基本1画面（Single-View Stage）」**で完結します。画面は以下の8大要素（Brand Areaから＋Buttonまで）が知的かつ幾何学的に配置されたコグニティブスペースです。

```
+───────────────────────────────────────────────────────────────────────────────────+
|  ① BRAND AREA (ブランド領域)                                 ③ AI STATUS        |
|  [Logo & OS Engine Active indicator]                         [Swarm CPU: 99.8%]   |
+───────────────────────────────────────────────────────────────────────────────────+
|                                                                                   |
|                                                                                   |
|                            ② MAIN INPUT (メイン入力)                              |
|                    ┌────────────────────────────────────────┐                     |
|                    │ 💭 どのような知的成功を具現化しますか？... │ ⑧ [+]                  |
|                    └────────────────────────────────────────┘                     |
|                                                                                   |
|                                                                                   |
+───────────────────────────────────────────────────────────────────────────────────+
|  ④ RECENT PROJECTS            ⑤ MEMORY                 ⑥ KNOWLEDGE   ⑦ AGENT      |
|  [ 📁 Active Enterprise LP ]  [ 👤 User DNA Schema ]  [ 🌐 Synced ] [ 👥 CTO/CMO ] |
|  [ 📁 Finance Audit Doc  ]  [ 🧠 Thinking: High  ]  [ 📚 42 Papers] [ 👥 CSO/CD  ] |
+───────────────────────────────────────────────────────────────────────────────────+
```

### ① ブランド領域 (Brand Area)
*   **配置**: 画面左上。極めて控えめで気品のあるタイポグラフィとシンボル。
*   **挙動**: カーソルを重ねると、現在接続されている「仮想企業OS（Intelligence OS）」の動作状態をインジケータ（Ambient Emerald Glow）で表示。

### ② メイン入力 (Main Input)
*   **配置**: 画面中央。すべての創造と意思決定が始まる「知的エンジンの点火プラグ」。
*   **意匠**: プレーンなinputボックスではなく、広めのpaddingを配し、薄いインナーシャドウと背景ぼかし（Backdrop Blur）を施した、まるで浮遊しているかのようなガラスモーフィズム（Glassmorphic Input Panel）。

### ③ AIステータス (AI Status Indicator)
*   **配置**: 画面右上。
*   **内容**: 現在裏側で待機・稼働しているAIエージェントの総スループット、ナレッジグラフの同期率、UQI平均（例: `UQI Target: 99.5%`）などを JetBrains Mono などの等幅フォントでミニマルに可視化（Technology Integrity）。

### ④ 最近のProject (Recent Projects Column)
*   **配置**: 画面下部、第1列。
*   **内容**: ユーザーが現在並行して推進しているプロジェクトのフォルダ。クリックすると、そのプロジェクトに適用されている「Project DNA」と過去のすべての制作物アセット（Reactコード、PDF、画像）が右からスライドインでステージ上に展開。

### ⑤ Memory (Memory Matrix Card)
*   **配置**: 画面下部、第2列。
*   **内容**: 現在蓄積されている「User DNA（思考、クリエイティブ美学、筆致スタイル）」の定着率をリアルタイムグラフ化して表示。AIがあなた個人をどれだけ理解しているかのシンクロ率（Sync Index）を提示。

### ⑥ Knowledge (Universal Knowledge Matrix)
*   **配置**: 画面下部、第3列。
*   **内容**: 同期済みのナレッジソース（Web検索グラウンディング、学術論文、公式API）の統合ネットワーク状態を、簡潔で洗練されたノード＆リンク（ドットパターン）のミニマリズムビジュアルで常時描写。

### ⑦ Agent (C-Suite Boardroom Overview)
*   **配置**: 画面下部、第4列。
*   **内容**: 現在チームとしてスタンバイしている主要AI専門役員（CEO, CTO, CMO, CD, Legal等）の稼働アイコンをリスト。ホバーすると、それぞれの役員AIの専門得意領域や評価成功率、現在のコンディションが段階的に開示。

### ⑧ ＋ボタン (Universal Creation Hub)
*   **配置**: メイン入力の右側、またはフローティング配備。
*   **思想**: このボタンこそが、ユーザーに対して「質問以上の知的創造」を提示するインターフェースのトリガー。クリックすると、円状またはシームレスなメニュー（Creative Radial Menu）が周囲にスプレッド展開。
*   **展開アセットメニュー（12種類）**:
    1.  **Document**: 企画書、要件定義書、契約書の自律起草。
    2.  **Presentation**: 美しく構造化されたスライドデッキ、ピッチデック。
    3.  **Image**: Creative AI による超高精細プロモーション用画像、ベクター。
    4.  **Video**: 動的な製品コンセプト動画、ブランドショートストーリー。
    5.  **Website**: 即時稼働するVite/React製ランディングページやポートフォリオ。
    6.  **App**: データベースと完全同期したフルスタックWebアプリケーション。
    7.  **SEO**: 検索エンジンでのオーガニック順位をハックするコンテンツ設計。
    8.  **MEO**: 地図検索で地域No.1を自律獲得するためのプロファイル構築。
    9.  **AIO**: GeminiやChatGPTに「世界最高のソリューション」として引用されるセマンティック最適化。
    10. **Research**: 学術論文、公式API、他社競合の徹底的なリバース調査。
    11. **Automation**: Slack, Stripe, データベースを連結する自律ワークフロー構築。
    12. **Agent**: ユーザー特有のタスクだけを行うための専用AI部下の作成。

## 4. UI/UX デザイン設計ルール (Strict Aesthetic Guidelines)

### 4.1 配色 ＆ マテリアル (Aesthetic Spectrum)

余計なテーマ選択を排除し、感性に直接訴えかける**「Cosmic Slate Theme (宇宙のスレート石)」**を極限まで美しく調整したライト＆ダークの両モードを完備。

#### A. ダークモード (Midnight Cosmos)
*   **ベース背景 (`bg-neutral-950`)**: 完全な漆黒に近いチャコールグレー。青みを絶妙に排し、眼精疲労を極限まで抑える。
*   **カード背景 (`bg-neutral-900/60` + `backdrop-blur-md`)**: 透過率60%の磨りガラスのようなスレート調。
*   **縁（Border） (`border-neutral-800/80`)**: 繊細な1pxのヘアライン。
*   **アクセント (`text-emerald-400` / `text-cyan-400`)**: 光ファイバーを想起させる、極小でありながら自己主張するエメラルド＆シアンの精密な光（Glow）。

#### B. ライトモード (Pristine Daylight)
*   **ベース背景 (`bg-neutral-50`)**: 雪原のような透き通ったオフホワイト。黄色みを極限まで排除。
*   **カード背景 (`bg-white/70` + `backdrop-blur-md`)**: 清潔感のある白。
*   **縁（Border） (`border-neutral-200/50`)**: 影を落とさずにオブジェクトの境界をシャープに定義。
*   **アクセント (`text-emerald-600` / `text-cyan-600`)**: 知性と清潔感を想起させる上品なディープエメラルド。

### 4.2 タイポグラフィ ＆ フォント (Typographic Rhythm)
*   **ディスプレイ / 見出し**: **Inter** または **Outfit**（洗練されたモダンさ、トラッキングは `-0.02em` のやや詰め気味で、文字同士が引き締まる美しさを表現）。
*   **ステータス / 数値 / ログコード**: **JetBrains Mono** または **Fira Code**（技術的正確性と信頼性を象徴する等幅フォント。大文字だけでトラッキング `0.1em` の広めのスペースを配し、SF的な格調高さを表現）。
*   **本文 / ガイドテキスト**: **Inter**（14px、`leading-relaxed` で行間を心地よく空け、目が滑るのを防止）。

### 4.3 空間・余白設計 (Intellectual White Space)
*   **「余白こそが最高のデザイン装飾である」**:  
    すべてのカードやインプット、ヘッダー、フッターは、密に詰め込まれず、全体の30%以上の十分なネガティブスペース（余白）に囲まれて配置されます。
*   **黄金パディング比**:  
    基本パディングは `p-6` (24px) または `p-8` (32px)。カード同士の間隔は `gap-6` (24px)。この均整のとれたスペーシングが、ユーザーに対して認知的安定感とラグジュアリーな高級感を与えます。

## 5. モーション ＆ アニメーション設計 (Fluid Motion Choreography)

動きはすべて、アニメーションライブラリ `motion` (import from `motion/react`) を通じて実装。CSSの単調なフェードインではなく、物理的なバネ（Spring Physics）の摩擦抵抗を感じる「オーガニックな応答」を再現します。

*   **ページイン・ staggered entrance (段階的登場)**:  
    起動時、ヘッダー、中央インプット、下部4列のカード、そして＋ボタンが、ミリ秒単位でずれて下から上にふわりと浮き上がってくるようなスタッガード（時間差）フェードインイージング。
*   **ホバー・レスポンス**:  
    カードにマウスホバーすると、1.5%だけサイズがスケールアップし（`scale: 1.015`）、ヘアライン境界がソフトに発光。クリックすると、バネのようにもっちりと一瞬縮むインタラクション。
*   **＋ボタン展開アニメーション (The Radial Burst)**:  
    ＋ボタンをクリックすると、周囲に12種類のメニューが、まるで花が開くように円を描いて、または美しいサイドドロワーとして、バネ（`damping: 15`, `stiffness: 120`）の弾性を伴って滑らかに飛び出し整列。

| :--- | :--- | :--- | :--- |
| **デスクトップ (Desktop)** | `lg:` (1024px〜) | 下部4列（Recent, Memory, Knowledge, Agent）が1列に横並びし、贅沢な余白を確保した黄金比レイアウト。 | メイン入力右側にエレガントな円形サークルメニュー、またはインプット下の洗練されたグリッドシートとして高速スプレッド。 |

## 7. 未来を感じる体験の定義 (The Future Assurance Spec)

> **「見た瞬間に未来を感じる」とは、昨日までのAIチャットが『古臭い道具』に見えてしまう、不可逆的な知的革命の体験である。**

*   **AI Visibility (知能の可視化)**:  
    ユーザーがインプットボックスに思考（テキスト）を1文字打ち込むたびに、AI Statusや各エージェント（CTO/CMO等）のアイコンの周囲に、小さなパルシングライトが明滅。「AIたちが、すでに裏側であなたの言葉を咀嚼し始めている」という動的な期待感を醸成。
*   **Ambient Music & Sound Response (微細な感性的レスポンス)**:  
    ボタンクリック、アセット生成の開始、UQI99点以上の承認が下りた瞬間に、極めて繊細で高解像度なアコースティック効果音（カチッというガラスの重なりや、ソフトな深音）がミリ秒で反応。
*   **The Blueprint Aura**:  
    アセットが完成する直前、カードの上に光のグリッド線が走り、建築図面（Blueprints）が描かれるようなワイヤーフレームアニメーションがホログラムのように明滅した後に、完成した美しいUIが姿を現す（Creation Stage）。

## 8. ファイナル・ステートメント (The Ultimate Standard)

Experience Bible Chapter 01が定義するSupreme Home Experienceは、デザインシステムの単なるカラーガイドを超え、AI Workspaceという「無限の知性を持つ電子企業」の玄関口を、世界で最も魅力的で心地よく、思考を吹き込みたくなる究極の聖域（Sanctuary）へと昇華させます。  
これに触れた瞬間、ユーザーは一切の説明を受けることなく、自らの知的生産性が新たな次元へと解き放たれたことを本能で確信します。
